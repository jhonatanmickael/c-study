# Challenge 01: Body Mass Index (BMI)

**Description**
Implementation of a program to calculate the Body Mass Index (BMI) based on an individual's mass (weight) and height. The solution focuses on precision and formatted output.

---

## Technical Specifications

### Input Format
* **Mass (Kg):** A floating-point value between 1 and 500.
* **Height (m):** A floating-point value between 1 and 2.8.

### Output Format
* The calculated BMI displayed as a floating-point number.
* The output is strictly rounded to **2 decimal places**.
