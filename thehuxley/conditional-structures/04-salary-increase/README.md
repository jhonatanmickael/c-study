# Salary Adjustment Challenge

A company decided to grant salary increases to its employees due to Labor Day. This program calculates the new salary based on the current earnings of an employee according to specific percentage brackets.

## Rules

The salary increase is applied based on the following criteria:

* Salaries **greater than 500.00** receive a **10%** increase.
* Salaries **greater than 300.00** (up to 500.00) receive a **7%** increase.
* All other salaries receive a **5%** increase.

## Input Format

* A single real number representing the current salary.
* Note: The program does not display messages during input.

## Output Format

* A float number formatted to exactly two decimal places.

## Examples

| Input | Output |
| :--- | :--- |
| 800 | 880.00 |
| 400 | 428.00 |
| 300 | 315.00 |
