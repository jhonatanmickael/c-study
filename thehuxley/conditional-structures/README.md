# Conditional Structures - Challenge Solutions

This directory contains solutions for programming challenges focused on decision-making structures hosted on [The Huxley](https://thehuxley.com/).

---

## Challenge List

* **01-quadratic-equation/**: A program that calculates the roots of a quadratic equation ($ax^2 + bx + c = 0$). It handles real roots based on the delta value.
* **02-championship/**: A program that determines the ranking between two football teams (Cormengo and Flaminthians) based on points (3 for win, 1 for draw) and goal difference.
* **03-increasing-order/**: A program that reads three integers and outputs them in ascending order, each on a separate line.
* **04-salary-increase/**: A program that calculates a salary reajustment based on three specific percentage brackets (5%, 7%, and 10%) depending on the current wage.
