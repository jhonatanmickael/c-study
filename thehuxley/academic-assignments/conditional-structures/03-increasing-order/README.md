# Increasing Order

## Description
Write a program that reads 3 integers and prints them in ascending order.

## Input Format
3 integers separated by a line break.

## Output Format
The 3 integers read, printed in ascending order, each on a new line.

## Examples

| Input | Output |
| :--- | :--- |
| 555 <br> 777 <br> 666 | 555 <br> 666 <br> 777 |
| 10 <br> 2 <br> 5 | 2 <br> 5 <br> 10 |
| -1 <br> -5 <br> 0 | -5 <br> -1 <br> 0 |
