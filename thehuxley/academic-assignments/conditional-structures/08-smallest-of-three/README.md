# Menor de Três Números (Smallest of Three)

Este programa faz parte de uma série de exercícios de lógica básica para o desenvolvimento de raciocínio condicional.

---

## Challenge Description (Descrição)

Faça um programa que leia **3 números inteiros** e imprima o **menor** deles.

| Nível (Level) | Estrela (Star) |
| :--- | :--- |
| Iniciante | ⭐☆☆☆☆ (1 de 5) |

---

## Input Format (Formato de Entrada)

* 03 números inteiros separados por um final de linha (um em cada linha).

## Output Format (Saída)

* O menor dos números digitados.

---

## Examples (Exemplos)

| Input (Entrada) | Output (Saída) |
| :--- | :--- |
| 1 <br> 1 <br> 1 | 1 |
| 10 <br> 5 <br> 8 | 5 |
| -2 <br> 0 <br> 4 | -2 |
