# Quadratic Equation

## Description
Every time Ambrósio calculates the roots of a quadratic equation, he forgets some detail and calculates it incorrectly. To avoid further mistakes, he decided to create a program that calculates the roots of a quadratic equation.

## Input Format
Consists of real numbers **a**, **b**, and **c**, corresponding to the coefficients of the quadratic equation ($ax^2 + bx + c = 0$).

## Output Format
* If the roots exist, it consists of two real numbers formatted to two decimal places.
* If the roots do not exist, the system must display the message: **NRR** (no real roots exist).
* If it is not a quadratic equation, the system must display the message: **NEESG** (not a quadratic equation).

## Examples

| Input | Output |
| :--- | :--- |
| 1 -5 6 | 3.00 <br> 2.00 |
| 1 1 5 | NRR |
| 0 2 10 | NEESG |
| 1 -10 25 | 5.00 <br> 5.00 |
