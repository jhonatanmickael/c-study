# Average Age

## Description
Write an algorithm to read an undetermined amount of data, each containing the age of an individual. The last reading (a negative age) should not be included in the calculations. The program must calculate and print the average age of this group.

## Input Format
The input contains an undetermined number of integers. The input stops when a **negative value** is read.

## Output Format
A single value corresponding to the average age of the individuals.
* The average must be printed with **two decimal places**.
* Must be followed by a line break.

## Examples

| Input | Output |
| :--- | :--- |
| 34 <br> 56 <br> 44 <br> 23 <br> -2 | 39.25 |
| 20 <br> 30 <br> -1 | 25.00 |
