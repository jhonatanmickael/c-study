# Factorial of a Sequence

## Difficulty
Level: 1 / 5 (Beginner)

---

## Description
Calculate the factorials of a given sequence of numbers. The factorial of a non-negative integer n, denoted by n!, is the product of all positive integers less than or equal to n. 

## Input Format
The program will receive a sequence of integers n, where 0 <= n <= 12.
The program ends its execution when the given number n is -1.

## Output Format
For each n, an integer k must be printed followed by a newline, corresponding to the resulting factorial.

---

## Examples

### Input
```text
5
3
-1

### Output
```text
120
6
