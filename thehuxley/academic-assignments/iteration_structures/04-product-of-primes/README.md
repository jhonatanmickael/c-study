# Product of Four Primes (The Huxley)

## Difficulty
* **Level:** ★☆☆☆☆ (1 out of 5 stars)

## Description
Given 04 integers, calculate the product between them if all 04 numbers entered are prime numbers. 
If any of them is not prime, print: SEM PRODUTO

> *This challenge was taken from The Huxley platform.*

## Input Format
* 04 integers separated by space.

## Output Format
* The result of the multiplication between them, if all 04 are prime.
* The message SEM PRODUTO, otherwise.

---

## Execution Examples

| Input | Output |
| :--- | :--- |
| 3 5 7 11 | 1155 |
| 2 4 5 7 | SEM PRODUTO |
| 1 2 3 5 | SEM PRODUTO |
