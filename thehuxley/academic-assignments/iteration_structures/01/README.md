# Sum of Elements

## Description
The objective is to write a program that calculates the sum of all elements provided in the input.

## Input Format
Consists of an integer **n** representing the number of elements to be read, followed by **n** integers, each on a new line.
* **Constraint**: $n \le 10000$.

## Output Format
Consists of a single integer indicating the total sum, followed by a line break.

## Examples

| Input | Output |
| :--- | :--- |
| 3 <br> 5 <br> 8 <br> 10 | 23 |
| 2 <br> 100 <br> -50 | 50 |
| 0 | 0 |
