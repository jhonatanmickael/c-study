# 06 - Rectangle Area

## Description
Write a program that calculates the area of a rectangle based on the lengths of its longer and shorter sides.

---

## Input Format
Two integers representing the sides of the rectangle. The values are guaranteed to be between 1 and 1000.

## Output Format
An integer representing the calculated area of the rectangle.

---

## Examples

| Input | Output |
| :--- | :--- |
| 5 <br> 10 | 50 |
| 1000 <br> 2 | 2000 |
| 2 <br> 3 | 6 |
