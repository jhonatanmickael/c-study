# The Huxley - Challenge Solutions

This directory contains solutions for programming challenges hosted on [The Huxley](https://thehuxley.com/). Each subfolder represents a specific problem solved.

---

##  Challenge List

The following challenges are classified as **Level 1** (Iniciante), focusing on basic logic and mathematical operators.

| Folder | Challenge | Level | Description |
| :--- | :--- | :---: | :--- |
| **01-imc** | BMI Calculation (Cálculo de IMC) | ⭐ (1) | Receives weight and height and returns the calculated index with rounding. |
| **02-comabem** | Eat Well (Coma Bem) | ⭐ (1) | Calculates the total bill including a 10% waiter fee (taxa de serviço). |
| **03-car-rental** | Car Rental (Aluguel de Carros) | ⭐ (1) | Calculates vehicle rental costs based on distance traveled and duration. |
| **04-last-digit** | Last Digit (Último Dígito) | ⭐ (1) | Extracts the units place of any integer, handling negative numbers correctly. |
| **05-salary-increase** | Salary Increase (Reajuste Salarial) | ⭐ (1) | Calculates a new salary after applying a fixed 7.5% increase. |
| **06-rectangle-area** | Rectangle Area (Área do Retângulo) | ⭐ (1) | Calculates the area given the side lengths of a rectangle. |
| **07-banknotes** | Banknotes (Cédulas) | ⭐ (1) | Decomposes a value into the minimum possible number of banknotes (100, 50, 20, etc.). |
| **08-basic-operations** | Basic Operations (Operações Básicas) | ⭐ (1) | Performs four basic operations on two real numbers with decimal precision. |

---

### 🛠️ Technologies and Concepts Practiced
These **Level 1** exercises consolidate the understanding of:
* Input and Output (**Entrada e Saída**).
* Arithmetic Operators and Modulo (**Operadores e Módulo**).
* String Formatting and Decimal Precision (**Formatação e Precisão**).
