# 02 - COMABEM Restaurant

## Description
Write a program that reads a value representing the amount spent by a customer at the **COMABEM** restaurant and prints the total amount to be paid, including a 10% service charge for the waiter.

---

## Input Format
The input consists of a real number (float) representing the amount spent by the customer.

## Output Format
The output consists of a real number formatted to two decimal places, representing the total amount including the 10% service charge.

---

## Examples

| Input | Output |
| :--- | :--- |
| 85 | 93.50 |
| 100.00 | 110.00 |
