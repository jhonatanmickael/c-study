# 04 - Algarismo da Casa das Unidades (Last Digit)

## Descrição
Escreva um programa que recebe um número inteiro como entrada e fornece o algarismo da casa das unidades deste número.

**Regras Específicas:**
* Se o número for positivo, a resposta é o algarismo positivo. (Ex: `3872` -> `2`)
* Se o número for negativo, o algarismo também deve ser negativo. (Ex: `-74` -> `-4`)

---

## Formato de Entrada
Um número inteiro ($n$).

## Formato de Saída
Um número inteiro representando o algarismo da casa das unidades.

---

## Exemplos

| Entrada | Saída |
| :--- | :--- |
| 45 | 5 |
| -74 | -4 |
| 3872 | 2 |
