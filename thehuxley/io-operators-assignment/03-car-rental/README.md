# Challenge 03: Car Rental Calculator

**Description**
Implementation of a rental cost calculator for "SAI DA FRENTE" car rental company. The program calculates the total amount due for a vehicle rental during the June festivities period, applying specific rates and a loyalty discount.

---

## Business Rules

* **Daily Rate:** R$ 30.00 per day.
* **Kilometer Charge:** R$ 0.01 per kilometer driven.
* **Promotional Discount:** 10% discount applied to the total rental value.

---

## Technical Specifications

### Input Format
* **Rental Duration (Days):** An integer value between 1 and 30.
* **Distance Traveled (Km):** A value between 1 and 1000.

### Output Format
* The final rental amount to be paid after the discount.
* The output is strictly rounded to **2 decimal places**.
