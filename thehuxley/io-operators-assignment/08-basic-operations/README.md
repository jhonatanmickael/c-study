# 08 - Basic Operations

## Description
You must implement a calculator that computes the 4 basic operations given 2 real numbers and prints the 4 results.

---

## Input Format
Two real numbers, one on each line.

## Output Format
4 real numbers, with 2 decimal places, representing the results of the 4 basic operations in the following order: sum, subtraction, multiplication, and division.

---

## Examples

| Input | Output |
| :--- | :--- |
| 1 <br> 2 | 3.00 <br> -1.00 <br> 2.00 <br> 0.50 |
