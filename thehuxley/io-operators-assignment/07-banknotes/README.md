# 07 - Banknotes

## Description
Read an integer value. Next, calculate the smallest possible number of banknotes (bills) into which the value can be decomposed. The banknotes considered are 100, 50, 20, 10, 5, 2, and 1. Finally, show the value read and the list of required banknotes.

---

## Input Format
The input file contains an integer value $N$ ($0 < N < 1000000$).

## Output Format
Print the read value, followed by the minimum quantity of each type of banknote required, exactly as shown in the example.

---

## Examples

| Input | Output |
| :--- | :--- |
| 576 | 576 <br> 5 nota(s) de R$ 100,00 <br> 1 nota(s) de R$ 50,00 <br> 1 nota(s) de R$ 20,00 <br> 0 nota(s) de R$ 10,00 <br> 1 nota(s) de R$ 5,00 <br> 0 nota(s) de R$ 2,00 <br> 1 nota(s) de R$ 1,00 |
