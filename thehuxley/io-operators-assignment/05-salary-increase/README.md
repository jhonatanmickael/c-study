# 05 - Salary Increase

## Description
Write a program that receives an employee's salary, calculates, and displays the new salary after a **7.5%** increase.

---

## Input Format
A real number representing the current salary.

## Output Format
A real number formatted to two decimal places representing the new salary after the increase.

---

## Examples

| Input | Output |
| :--- | :--- |
| 100.00 | 107.50 |
| 2000.00 | 2150.00 |
| 1500.50 | 1613.04 |
