# Question 25: Product of Integers

## Prompt
25) Repetition. Read integer values. Next, calculate the product of these values and assign this operation to the variable PROD. Finally, display the PROD variable with the corresponding message.

## Test Examples
| Input (Values) | Expected Output |
| :--- | :--- |
| 3, 5, 2 | PROD = 30 |
| 10, 10 | PROD = 100 |
| 1, 2, 3, 4 | PROD = 24 |
| 5, 0, 10 | PROD = 0 |
