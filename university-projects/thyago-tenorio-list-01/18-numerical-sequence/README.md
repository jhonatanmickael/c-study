# Exercise 18: Numerical Sequence from 1 to 1000

## Description
The objective of this exercise is to develop a program that uses repetition structures to display all integers, starting at 1 and ending at 1000.

---

## Input Format
This program does not require any data input from the user.

## Output Format
The output must present the sequence of integers from 1 to 1000, with one number per line to facilitate reading.
