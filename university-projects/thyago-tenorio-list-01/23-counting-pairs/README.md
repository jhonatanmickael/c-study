# Question 23: Repetition and Decision

## Statement
Lucas is learning how to sum, but he still makes a lot of mistakes, so he wants you to help him. The problem is as follows: Lucas wants to know how many pairs (X, Y) exist such that:

* A ≤ X ≤ B
* C ≤ Y ≤ D
* X + Y > K

## Input
1 3 1 3 4

## Output
3
