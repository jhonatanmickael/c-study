# Question 16 - Quadratic Equation (Bhaskara)

## Statement
Write a program that calculates the roots of a quadratic equation, given the integers a, b, and c. Print an error message if no real roots exist.

## Formula
1. **Delta:** $\Delta = b^2 - 4ac$
2. **Roots:** $x = \frac{-b \pm \sqrt{\Delta}}{2a}$

## Test Cases
| Input (a, b, c) | Output |
| :--- | :--- |
| 1, -5, 6 | x1 = 3.00, x2 = 2.00 |
| 1, -4, 4 | x1 = 2.00, x2 = 2.00 |
| 1, 1, 1 | Error: Delta is negative (no real roots) |
