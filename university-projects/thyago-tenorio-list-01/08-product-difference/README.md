# Question 08 - Arithmetic Operations

## Problem Statement
Read four integer values (A, B, C, and D). Calculate and display the difference between the product of A and B and the product of C and D.

## Requirements
1. The program must read four integer inputs.
2. Formula: Result = (A * B) - (C * D).
3. **Note:** This implementation considers the **absolute value** of the difference (the result will always be positive).

## Project Structure
* **q08.c**: Source code containing the program logic.
* **main**: Compiled executable binary.

## Execution Example
Input:
5, 6, 7, 8

Output:
Difference (Absolute): 26
