# Question 02 - Assignment

## Problem Statement
You are organizing an event and want your guests to feel important. When each guest arrives at the venue, you ask for their name and type it into the computer. Their name is then displayed on a luminous panel at the hall entrance. The message displayed is: "Seja muito bem-vindo Fulano de Tal", where "Fulano de Tal" is the name of the person who arrived.

## Requirements
1. The program must read a string (text) representing the guest's name.
2. The output must strictly follow the text pattern established in the problem statement.
3. Use standard functions from the stdio.h library for data input and output.

## Project Structure
* **q02.c**: Source code containing the program logic.
* **main**: Compiled executable binary.

## Execution Example
Input:
Jhonatan Mickael

Output:
Seja muito bem-vindo Jhonatan Mickael
