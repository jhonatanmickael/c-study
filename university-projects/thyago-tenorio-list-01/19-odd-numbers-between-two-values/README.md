# Exercise 19: Odd Numbers Between Two Given Numbers

## Description
The objective of this exercise is to develop a program that receives two integer numbers as input and uses repetition structures to display all odd numbers located between them.

---

## Input Format
Two integers representing the start and the end of the interval.

## Output Format
The output must present all odd numbers within the given range, one per line or separated by spaces.
