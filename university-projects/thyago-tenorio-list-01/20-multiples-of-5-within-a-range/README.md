# Exercise 20: Multiples of 5 Within a Range

## Description
The objective of this exercise is to develop a program that reads two integers representing the start and end of an interval. The program must then use decision and repetition structures to identify and print all multiples of 5 within that specific range.

---

## Input Format
Two integers representing the start and the end of the interval.

## Output Format
The output must present all numbers that are multiples of 5 within the given range, one per line or separated by spaces.
