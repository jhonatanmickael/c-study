# Question 11 - Decision Structures (Smallest of Three)

## Problem Statement
Write a program that reads three integers from the keyboard and prints the smallest value among them.

## Project Structure
* **q11.c**: Source code containing the decision logic (`if/else`).
* **main**: Compiled executable binary.

## Validation Test Cases
| Inputs (A, B, C) | Expected Result |
| :--- | :--- |
| 5, 10, 15 | 5 |
| 80, 45, 12 | 12 |
| 10, -50, 20 | -50 |
| 7, 7, 10 | 7 |
