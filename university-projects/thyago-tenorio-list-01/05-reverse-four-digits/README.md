# Question 05 - Arithmetic Operations

## Problem Statement
Create a program that reverses a four-digit integer provided by the user.

## Requirements
1. The program must read an integer containing exactly four digits.
2. It must process each digit individually to reverse their order.
3. The output must display the resulting inverted number.

## Project Structure
* **q05.c**: Source code containing the program logic (using arrays and loops).
* **main**: Compiled executable binary.

## Execution Example
Input:
1234

Output:
4321
