# Question 04 - Arithmetic Operations

## Problem Statement
Create a program that reverses a two-digit integer provided by the user.

## Requirements
1. The program must read an integer containing exactly two digits.
2. It must isolate the tens and units digits using arithmetic operations (division and modulo).
3. The output must display the digits in reverse order.

## Project Structure
* **q04.c**: Source code containing the program logic.
* **main**: Compiled executable binary.

## Execution Example
Input:
42

Output:
24
