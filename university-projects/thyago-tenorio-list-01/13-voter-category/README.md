# Question 13 - Voter Category

## Statement
Write a program that reads a person's age (integer) and informs their voting category:
- Non-voter (under 16 years old)
- Mandatory voter (18 to 65 years old, inclusive)
- Optional voter (between 16 and 18 years old, or over 65 years old)

## Test Cases
| Input | Output |
| :--- | :--- |
| 15 | Non-voter |
| 17 | Optional voter |
| 25 | Mandatory voter |
| 70 | Optional voter |
