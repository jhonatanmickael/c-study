# Question 22: Average of N Real Numbers

## Prompt
22) Repetition. Write a program that calculates the average of n given real numbers.

| 2 | 5.5, 4.5 | Average: 5.00 |
| 4 | 1.0, 1.0, 1.0, 1.0 | Average: 1.00 |
| 5 | 12.5, 13.2, 11.0, 14.8, 10.5 | Average: 12.40 |
