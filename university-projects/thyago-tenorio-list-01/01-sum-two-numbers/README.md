# Question 01 - Assignment

## Problem Statement
You are organizing a bingo game and want each drawn number to appear on the screen. Create a program that, given a typed number, prints: "O número sorteado foi: X", where X is the drawn number.

## Requirements
1. The program must read an integer representing the drawn number.
2. The output must strictly follow the text pattern established in the problem statement.
3. Use standard functions from the stdio.h library for data input and output.

## Project Structure
* **q01.c**: Source code containing the program logic.
* **main**: Compiled executable binary.

## Execution Example
Input:
25

Output:
O número sorteado foi: 25
