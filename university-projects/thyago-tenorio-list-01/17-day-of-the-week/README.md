# Question 17 - Day of the Week

## Statement
Write a program that reads an integer from the user and returns the corresponding day of the week in full (e.g., 1 for Sunday, 2 for Monday, etc.). If the number is out of the 1-7 range, print an error message.

## Mapping
| Number | Day of the Week |
| :--- | :--- |
| 1 | Sunday |
| 2 | Monday |
| 3 | Tuesday |
| 4 | Wednesday |
| 5 | Thursday |
| 6 | Friday |
| 7 | Saturday |
| Others | Invalid Day |
