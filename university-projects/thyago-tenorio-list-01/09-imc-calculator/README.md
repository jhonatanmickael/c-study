# Question 09 - Arithmetic Operations (BMI Calculator)

## Problem Statement
Create a program that calculates a person's Body Mass Index (BMI/IMC) and classifies them according to the result.

## Formula
The formula used is:
$$IMC = \frac{P}{A \times A}$$
Where:
* **P**: Weight (Peso) in kilograms.
* **A**: Height (Altura) in meters.

## Requirements
1. The program must read the weight (float) and height (float).
2. It must calculate the IMC using the provided formula.
3. It must classify the result based on standard IMC tables (Underweight, Normal, Overweight, Obesity).

## Project Structure
* **q09.c**: Source code containing the logic with `if/else` structures.
* **main**: Compiled executable binary.

## Execution Example
Input:
Weight: 70.0
Height: 1.75

Output:
IMC: 22.86
Classification: Normal Weight
