# Question 15 - Month of the Year

## Statement
Write a program that reads an integer and informs the corresponding month of the year in full. If it is an invalid month, inform the user.

## Test Cases
| Input | Output |
| :--- | :--- |
| 1 | January |
| 6 | June |
| 12 | December |
