# Question 07 - Arithmetic Operations

## Problem Statement
Create a program that manages the banknote output of an ATM. Given a specific amount, the program must calculate the minimum number of banknotes required to fulfill the withdrawal.

## Requirements
1. The program must follow the "optimal distribution" criterion (the smallest possible number of bills).
2. Available banknotes: R$ 50, R$ 20, R$ 10, R$ 5, and R$ 1.
3. The input is an integer representing the total amount to be withdrawn.
4. The output must list the quantity of each specific banknote.

## Project Structure
* **q07.c**: Source code containing the program logic.
* **main**: Compiled executable binary.

## Execution Example
Input:
188

Output:
50: 3
20: 1
10: 1
5: 1
1: 3
