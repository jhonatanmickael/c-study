# Question 10 - Arithmetic Operations (Tile Calculator)

## Problem Statement
Calculate how many tiles are needed to tile a wall. It is necessary to know the wall's height (AP) and width (LP), as well as the tile's height (A) and width (LA). Read the data from the keyboard.

## Project Structure
* **q10.c**: Source code containing the program logic.
* **main**: Compiled executable binary.

## Validation Test Cases
| Wall (H x W) | Tile (H x W) | Expected Result |
| :--- | :--- | :--- |
| 3.0m x 4.0m | 0.5m x 0.5m | 48 tiles |
| 2.5m x 3.0m | 0.2m x 0.2m | 187.5 tiles |
