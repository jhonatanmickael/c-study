# Question 14 - Student Status (Average Grades)

## Statement
Write a program that reads three student grades (real values), calculates their arithmetic average, and prints a message stating if the student was approved, failed, or must take the final exam. The criteria are:
- Approved (average >= 7)
- Failed (average < 3)
- Final Exam (3 <= average < 7)

## Test Cases
| Input (Grades) | Average | Status |
| :--- | :--- | :--- |
| 8.0, 7.5, 9.0 | 8.16 | Approved |
| 2.0, 1.5, 3.0 | 2.16 | Failed |
| 5.0, 6.0, 4.0 | 5.00 | Final Exam |
