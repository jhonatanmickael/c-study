# Question 03 - Arithmetic Operations

## Problem Statement
Create a program that reads an integer and prints its predecessor (the previous integer) and its successor (the following integer).

## Requirements
1. The program must read a single integer value from the user.
2. It must calculate the predecessor by subtracting 1 and the successor by adding 1.
3. Both results must be displayed clearly in the output.

## Project Structure
* **q03.c**: Source code containing the program logic.
* **main**: Compiled executable binary.

## Execution Example
Input:
10

Output:
Predecessor: 9
Successor: 11
