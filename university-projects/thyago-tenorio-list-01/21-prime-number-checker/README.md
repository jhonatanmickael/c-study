# Question 21: Prime Number Checker

## Prompt
21) Repetition and Decision. Write a program that, given a number, indicates whether it is a prime or not.

## Test Examples
| Input | Expected Output |
| :--- | :--- |
| 1 | Not prime. |
| 2 | Prime. |
| 9 | Not prime. |
| 169 | Not prime. |
| 2147483647 | Prime. |
