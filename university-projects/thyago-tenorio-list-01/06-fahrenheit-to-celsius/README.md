# Question 06 - Arithmetic Operations

## Problem Statement
Create a program that converts a given temperature in degrees Fahrenheit to degrees Celsius.

## Requirements
1. The program must read a floating-point value representing the temperature in Fahrenheit.
2. It must apply the conversion formula: C = (F - 32) * 5 / 9.
3. The output must display the result in Celsius, preferably formatted to two decimal places.

## Project Structure
* **q06.c**: Source code containing the program logic.
* **main**: Compiled executable binary.

## Execution Example
Input:
212

Output:
Temperature in Celsius: 100.00
