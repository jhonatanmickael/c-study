# Question 28: Blobs Food Supply

## Statement
In planet Alpha lives a creature called Blobs, which eats exactly half of its available food supply every day. Write an algorithm that reads the initial food capacity (in Kg) and calculates how many days will pass before the supply reaches 1 kg or less.

## Input Example
100.0

## Output Example
7 dias
