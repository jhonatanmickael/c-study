# List 01 - Introduction to Programming

This directory contains the first set of exercises for the Introduction to Programming course, taught by Professor Thyago Tenório at the Federal University of Alagoas (UFAL).

## Objectives
The primary goal of this list is to practice basic C programming concepts, including:
* Variable assignment and memory storage.
* Standard Input/Output (scanf and printf).
* Basic arithmetic operations and digit manipulation.

## Exercises Summary
Each folder below contains the source code (qXX.c), the compiled executable (main), and a specific README with the problem statement.

1. [Question 01 - Bingo Announcement](./01-bingo-announcement/)
   * Topic: Basic assignment and output.
2. [Question 02 - Guest Welcome](./02-guest-welcome/)
   * Topic: String handling and user interaction.
3. [Question 03 - Predecessor and Successor](./03-predecessor-successor/)
   * Topic: Simple arithmetic calculations.
4. [Question 04 - Reverse Two Digits](./04-reverse-two-digits/)
   * Topic: Digit isolation using modulo and division.
5. [Question 05 - Reverse Four Digits](./05-reverse-four-digits/)
   * Topic: Advanced digit manipulation and array logic.
6. [Question 06 - Fahrenheit to Celsius](./06-fahrenheit-to-celsius/)
   * Topic: Floating-point arithmetic and temperature conversion.
